# 🌦️ Weather App

A simple and responsive Weather App built using **HTML, CSS, and JavaScript**. 
This application fetches real-time weather data from the OpenWeather API and displays current weather information for 
any city entered by the user.

## 📖 Features

- 🌍 Search weather by city name
- 🌡️ Display current temperature
- ☁️ Show weather condition
- 💧 Display humidity level
- 💨 Show wind speed
- ⌨️ Search using the Enter key or Search button
- 📱 Responsive design for all devices
- ⚡ Fast weather data fetching using asynchronous JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

---

## 🔑 API

This project uses the **OpenWeather API** to retrieve live weather information.

To use your own API key:

1. Create an account on **OpenWeather**.
2. Generate your API key.
3. Replace the existing API key inside the JavaScript file with your own.

Example:

```javascript
const API_KEY = "YOUR_API_KEY";
```
